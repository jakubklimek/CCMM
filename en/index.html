full address

Profiled relationship IRI(s) locn:fullAddress
IRI https://model.ccmm.cz/research-data/Address.fullAddress
Label full address
Definition The complete address written as a string.
Domain Address
Range xsd:string [0..*]
Profiles
  • property full address (locn:fullAddress)
    Definition:
Usage note Use of this property is recommended as it will not suffer any misunderstandings that might arise through the breaking up of an address into its component parts. This property is analogous to vCard's label property but with two important differences: (1) formatting is not assumed so that, unlike vCard label, it may not be suitable to print this on an address label, (2) vCard's label property has a domain of vCard Address; the fullAddress property has no such restriction. An example of a full address is "Champ de Mars, 5 Avenue Anatole France, 75007 Paris, France".

locator name

Profiled relationship IRI(s) locn:locatorName
IRI https://model.ccmm.cz/research-data/Address.locatorName
Label locator name
Definition Proper noun(s) applied to the real world entity identified by the locator. The locator name could be the name of the property or complex, of the building or part of the building, or it could be the name of a room inside a building.
Domain Address
Range xsd:string [0..*]
Profiles
  • property locator name (locn:locatorName)
    Definition:
Usage note The locator name could be the name of the property or complex, of the building or part of the building, or it could be the name of a room inside a building. The key difference between a locator designator and a locator name is that the latter is a proper name and is unlikely to include digits. For example, "Shumann, Berlaymont" is a meeting room within the European Commission headquarters for which locator name is more appropriate than locator.
